# **App Name**: Skill Showcase

## Core Features:

- Profile Display: Muestra tu información profesional de forma clara:

- Foto de perfil optimizada para web

- Título profesional (Software Developer / Backend .NET Developer)

- Resumen breve y profesional
- Tech Stack Presentation: Una sección visual y limpia donde se destacan tus tecnologías:

- Iconos minimalistas y consistentes (SVG)

- Descripción corta de cada tecnología que dominas

- Opcional: Nivel visual (e.g., barras o “proficient / advanced”)
- Experience Timeline: Línea de tiempo elegante para mostrar:

- Experiencia profesional

- Voluntariado

- Proyectos destacados (si quieres añadirlos)
- CV Download: Botón prominente para descargar tu CV en PDF:

- Archivo alojado en /public/cv.pdf

- Traza de clics opcional con Vercel Analytics (más adelante)
- External Links: Enlaces limpios con iconos a:

- LinkedIn

- GitHub

- Email / Contacto
- SEO Optimized: Incluye lo necesario para que Google te indexe perfectamente:

- Metadatos correctos en cada página

- OpenGraph + Twitter Cards

- Sitemap auto-generado por Next.js

- JSON-LD para perfil profesional (schema.org Person)

- Rutas limpias y rendimiento alto

## Style Guidelines:

- Primary color: Dark navy blue — #243A73

Profesional, serio, corporativo.
- Background color: Light gray — #F0F4F8

Neutro y elegante.
- Accent color: Muted purple — #635985

Un toque moderno sin romper la estética.
- Headline font: Space Grotesk (moderno, profesional)
- Body font: Inter (estándar en diseño moderno)
- Iconos minimalistas (preferiblemente outline).

Todos del mismo pack (e.g., Tabler Icons, simple-icons para tech).
- Estructura grid-based con mucho espacio blanco.

Secciones separadas y fácilmente navegables.

Mantener consistencia en espaciados (Tailwind: px-6, py-12, gap-8).
- Sutiles, profesionales:

- Hover effects en tarjetas e iconos

- Smooth scrolling entre secciones

- Animaciones leves con framer-motion (si quieres añadirlo más adelante)
